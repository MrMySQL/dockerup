usage:
    SYNOPSIS
           dockerup.sh [<options>] -t <ticket number or version number> -i <docker image version> [<run phpstorm>]

    DESCRIPTION
            Prepare environment for work.
            Frontend link:   http://127.0.0.1:XXXX0, where XXXX is ticket number
            Mailer link:     http://127.0.0.1:XXXX1

    OPTIONS
            -h|--help
                Show HELP.

            <ticket number or version number>
                Use only integer value.
                    Example:
                        1234 is number of ticket MDVA-1234
                        213  is version of Magento
                Will be used for ports naming.
                    EXAMPLE:    12340, 2130

            <docker image version>
                Docker image version.
                Available versions see on https://docker.sparta.corp.magento.com:5000/v2/m2/tags/list
                    EXAMPLE:    2.1.7-git-sd

            <run phpstorm>
                -o
                    [!Developing] Run PHPStorm program.
